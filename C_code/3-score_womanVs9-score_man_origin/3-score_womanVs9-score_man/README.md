#### 保留原有颜值计算逻辑

#### 财产评分系统

#### 虚拟人物卡片

#### 匹配度进度条

#### 详细评分统计

## 使用说明

### 安装依赖：

    sudo apt-get install libmicrohttpd-dev # Ubuntu/Debian

    brew install libmicrohttpd  # macOS
### 运行项目：

	chmod +x run.sh

	./run.sh
或者

    bash run.sh
### 使用流程：
#### 填写个人信息

#### 设置颜值和财产条件

#### 开始给出表白人员信息

#### 查看匹配结果

#### 可以重新设置条件或重新开始
